- Product name: `Sardine`
- Chinese name: `娌欎竵楸煎帇缂ー
- Repository: `sardine-ios`

## Positioning

Sardine is a small, focused utility. It compresses large homework videos into smaller files while keeping text readable.

Brand memory point:

> A sardine packs big video into a small tin, but keeps the homework readable.

## Relationship to Marlin

Marlin is the earlier ocean-themed network monitoring project. Sardine should feel like it belongs to the same product family:

- ocean animal name;
- simple mascot silhouette;
- calm teal / navy palette;
- practical utility, not flashy consumer branding.

Sardine should not copy Marlin鈥檚 billfish shape. It should use a compact sardine silhouette with clean side stripes.

## Image generation prompt

The avatar was generated with the built-in image generation tool using Marlin鈥檚 existing mascot as style reference.

Prompt summary:

```text
Create a square app avatar for an iOS video compression app named Sardine.
Use a compact sardine silhouette, facing right, centered inside a warm cream circle on a soft teal background.
Style should feel related to the Marlin mascot: simple ocean animal silhouette, dark navy-to-teal gradient, crisp vector-like edges.
Add two or three clean side stripes suggesting compressed video lines.
No text, no letters, no watermark, no marlin, no swordfish bill, no extra objects.
```
